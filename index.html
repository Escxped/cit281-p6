🧩 Project 6 – Full Stack API Application

This project builds a complete web application using Express.js and Node.js to create a backend API that serves and manages data. The frontend interacts dynamically with the server using fetch calls. A technical report also documents the system design and implementation process.

🔧 Features

  • RESTful API endpoints with Express
  • JSON data processing and routing
  • Frontend integration using fetch()
  • Modular file structure and reusable logic

📁 Key Files

  • p6-server.js – Express server logic
  • package.json – Dependency management
  • public/ – Frontend files (HTML, CSS, JS)
  • p6.pdf – Project report / technical write-up

🚀 How to Run

  1. Install packages: npm install
  2. Run server: node p6-server.js
  3. Open browser to: http://localhost:3000
